commons.db is used to connect with database. For increasing accuracy .math file is a module of calculation to proceed the add and subtract.

DAO file is also connected with database which defines the ports and IAccountdao file is implementation class.

domain file defines the service logic layer which achieve the function of login, logout and check the balance.

web file including manage the page jumps and redirect the page. 

wsclient file: BOC distribute the web service with WDS file and then generate the BOC client code at the side of BOA. So that the local 
host can invoke the client code with long-distance.

In addition, the webRoot document includes jsp file which defines the UI of balance,change password, login and etc.

lib file is a whole description of the web service, includeing every port direction
